# 就活管理アプリ 実行方法

## ✅ 解決済み：アプリが正常に起動します！

## 簡単な起動方法

### 方法1: バッチファイルから起動（推奨）

1. **`start.bat`** をダブルクリック
2. 自動的にビルドと起動が行われます
3. ウィンドウが表示されます！

### 方法2: コマンドラインから起動

1. コマンドプロンプトまたはPowerShellを開く
2. このフォルダに移動
3. 以下のコマンドを実行:

```bash
npm run build
npm start
```

## 初回セットアップ（重要！）

**初めて起動する場合は、必ず以下を実行してください：**

```bash
npm install
```

これにより、必要なパッケージがすべてインストールされます。

## Slack通知の設定方法

1. アプリを起動
2. 「設定」ページに移動
3. Slack Webhook URLを入力
   - Slackワークスペースで「Incoming Webhooks」アプリを追加
   - 通知を受け取りたいチャンネルを選択
   - 生成されたWebhook URLをコピーして貼り付け
4. 「自動通知を有効にする」をONに設定
5. 「保存」をクリック

これで、イベントのリマインダー時刻に自動的にSlackへ通知されます。

## データベース

- アプリケーションのデータは `C:\Users\<ユーザー名>\AppData\Roaming\job-hunting-manager\job-hunting.db` に保存されます
- すべてローカルに保存されるため、オフラインでも使用可能です

## トラブルシューティング

### アプリが起動しない場合

1. Node.jsがインストールされているか確認
   ```bash
   node --version
   ```
2. 依存関係を再インストール
   ```bash
   npm install
   ```
3. キャッシュをクリア
   ```bash
   npm cache clean --force
   npm install
   ```

### ビルドエラーが発生する場合

```bash
npm run build:main
npm run build:renderer
```

個別にビルドして、どちらでエラーが発生しているか確認してください。
